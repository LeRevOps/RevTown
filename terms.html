The service

LeClaw provides a RevOps agent framework that connects to CRM platforms (HubSpot, Salesforce) and runs automated checks against your data. The hosted dashboard (app.leclaw.io) runs agents on a schedule and delivers results via the dashboard and Slack. The open source CLI (npx leclaw) runs agents on your own machine using your own credentials.

Shadow mode and write-back

LeClaw operates in shadow mode by default. This means agents read your CRM data but do not modify, delete, or create any records unless you explicitly enable write-back mode. Write-back mode is a paid feature that requires deliberate activation. You will always be informed before any write operation is performed.
